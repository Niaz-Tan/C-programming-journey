# What is C?

## Definition

**C** is a **general-purpose, procedural, compiled programming language** designed to create efficient and high-performance software.

It allows programmers to write instructions that are translated directly into machine code, making programs execute very quickly.

---

## Why is C called a Programming Language?

A programming language is a medium used to communicate instructions to a computer.

C provides a set of rules (syntax) and keywords that allow programmers to write these instructions in a human-readable form.

Example:

```c
#include <stdio.h>

int main()
{
    printf("Hello, World!");
    return 0;
}
```

The compiler translates this code into machine language so the CPU can execute it.

---

## Key Characteristics

- General-purpose language
- Procedural programming language
- Compiled language
- Statically typed
- Case-sensitive
- Provides direct memory access through pointers
- Efficient and portable

---

## Why is C Considered Fast?

Unlike interpreted languages, C code is compiled into machine code before execution.

```text
C Source Code (.c)
        │
        ▼
     Compiler
        │
        ▼
 Machine Code
        │
        ▼
   Executed by CPU
```

Since the CPU executes machine code directly, C programs have very little runtime overhead.

---

## Summary

C is a compiled, procedural programming language that enables developers to write efficient and portable software. It gives programmers fine control over system resources, making it one of the fundamental languages in computer science.
